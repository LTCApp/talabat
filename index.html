🛒 استخراج منتجات سوق الجملة

أداة لاستخراج جميع منتجات سوق الجملة من موقع طلبات مع الأكواد والأسعار والصور

جاري فحص الخادم...
🎆 مراكز المساعدة والإرشادات

🔍 طريقة الاستخدام:

  1. اختر المتجر من القائمة أو أدخل رابطاً مخصصاً
  2. اضغط "استخراج المنتجات"
  3. انتظر حتى اكتمال العملية
  4. راجع النتائج وصدرها بصيغة CSV/JSON

⚠️ إذا لم يعمل الاستخراج:

  • تأكد من تشغيل الخادم: npm start
  • استخدم تطبيق Python: python extractor.py
  • تحقق من صحة الرابط

💻 للمطورين:

API Endpoint: POST /api/extract
